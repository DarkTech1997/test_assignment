# mask_finder.py

Поиск минимальной маски подсети между двумя IP-адресами.  
Поддерживает IPv4 и IPv6.

## Установка

Скопируйте файл `mask_finder.py` в любую папку.

## Использование

python mask_finder.py <IP1> <IP2>

## Примеры

python mask_finder.py 192.168.1.1 192.168.2.1

Вывод: /22

python mask_finder.py 192.168.1.1 192.168.1.1

Вывод: /32

python mask_finder.py 10.0.0.1 10.0.0.0

Вывод: /31

python mask_finder.py 1.1.1.1 200.200.200.200

Вывод: /0

python mask_finder.py 2001:db8::1 2001:db8::2

Вывод: /127

python mask_finder.py ::1 ::2

Вывод: /126

python mask_finder.py :: ::1

Вывод: /127

python mask_finder.py 192.168.1.1 2001:db8::1

Error: IP addresses of different versions (IPv4 and IPv6)

python mask_finder.py 256.1.1.1 192.168.1.1

Error: Octet out of range 0-255: 256

## Коды возврата

0 - успешное выполнение
1 - Ошибка в аргументах коммандной строки
2 - Ошибка формата или не совместимые версии

## Как это работает

1. Определяется версия каждого IP (4 или 6) по наличию точки или двоеточия.
2. IP-адреса преобразуются в целые числа:
- IPv4 → 32-битное число (каждый октет сдвигается на 8 бит)
- IPv6 → 128-битное число (каждая группа сдвигается на 16 бит)
3. Вычисляется XOR двух чисел — биты, где адреса различаются.
4. Находится позиция самого старшего единичного бита (diff.bit_length()).
5. Длина маски = общее количество бит − позиция старшего бита.
6. Если адреса совпадают, маска равна общему количеству бит.

## Требования

- Python 3.6 или выше
- Стандартная библиотека

## Структура кода

detect_version(ip) - Определяет версию IP (4, 6 или 0)

ip_to_int(ip, version) - Преобразует IP в целое число

main() - Обрабатывает аргументы, вызывает функции, выводит результат
